# Supplementary data bundles

Self-contained data bundles that supplement the main report and the
public article but are not part of the headline reproduction.

## Available bundles

- **`oil_iran_word_matrix/`** -- 30-word co-occurrence matrix of
  oil- and Iran-related vocabulary across the 73-day window, with
  build / render / verify pipeline and preview PNGs. Underlies the
  supplementary word-matrix visual in the article.

Each bundle is self-contained and runs via its own `run.sh`.
